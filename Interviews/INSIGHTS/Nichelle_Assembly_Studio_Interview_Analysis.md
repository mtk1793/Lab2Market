## Company Overview: The Assembly Studio

### Status:
- **Founded:** October 2025 (pilot phase)
- **Stage:** Very early - only 4 months old
- **Focus:** Secure digital inventory platform for Canadian manufacturing

### Core Value Proposition:
1. **Digital Inventory:** Convert physical warehouse stock to digital design files
2. **IP Security:** Secure warehousing and transport of design files (their differentiator)
3. **Canadian Ecosystem Building:** Addressing 10-year lag behind Europe/US

### What They're Building:

**Technology Stack (Dustin's Design):**
- **Local Hardware:** Raspberry Pi 5 boards at each manufacturing site (3-4 boards per site)
- **Encrypted Storage:** Files stored encrypted on-site at manufacturing location
- **Physical Pass Keys:** Highest security level for sensitive IP
- **No Cloud Strategy:** Avoiding US/Canadian/Chinese clouds for security
- **Decentralized Architecture:** Each site has local hardware; redundancy built in
- **AI Integration (Future):** Raspberry Pi 5 AI hat for visual QA, failure detection
- **Automation Focus:** Robots, automated plate handling, preventive maintenance scheduling

**Security Model:**
- Multiple security levels (some clients don't care, others require maximum protection)
- Encrypted file transfer between sites
- **Goal:** "A human cannot access the file" - fully automated secure workflow
- Currently no encrypted path from local server â†’ printer (identified gap)
