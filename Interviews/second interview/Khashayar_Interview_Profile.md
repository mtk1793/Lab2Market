## Process & Workflow Insights

### Testing Process (Detailed)

**Step 1: Material Receipt & Microstructure Analysis**
- Receive material (powder-based)
- Analyze microstructure
- Compare with traditional manufacturing methods
- Document differences

**Step 2: Corrosion Testing**
- Alloy-specific tests designed based on material properties
- Previous behavior analysis
- Current test design
- Both destructive and non-destructive testing

**Step 3: Material Optimization**
- Apply modifications (heat treatment, processing)
- Test if changes improve quality
- Iterative optimization

**Step 4: Reporting & Decision**
- Generate comprehensive report
- Present to supervisors and industrial partners
- Decision: Accept material or decline

### Confidential External Testing

> *"At some point they will send the products while they are black box to other universities and ask them for just some tests nothing more as everything is confidential finally we do the evaluation by ourselves we only need their instruments."*

**Process:**
- Send materials to other universities (black box)
- Request specific tests only
- No context or details shared
- Receive data back for internal analysis
- Use external instruments when not available in-house

**Why This Matters:**
- Even within academic networks, confidentiality is paramount
- Labs need access to diverse instrument sets
- Evaluation happens internally (trust in own analysis)
